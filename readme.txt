# Multi Pages Form Validation from HTML, CSS and JavaScript

Simple client side form validation. Check length, email, radio button validation

## Project Specifications

- Create form UI
- Multi Pages Form
- check if all the inputs fields are filled 
- Show alert messages for specific inputs
- unable to move to next pages if the form is not valid
- check if atleast one the check boxes is selected or not
- check if user gave the proper email or not 
- check validation for Mobile number 
- user can see summary of information before submitting
